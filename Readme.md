This is a folder for testing the script `~/Documents/Scripts/convert_chatgpt_to_obsidian_copilot.py` that converts ChatGPT data export to Markdown notes readable by Obsidian Copilot pluginas conversations. In that way it's possible to transfer and continue all your conversations locally in Obsidian with Copilot plugin.


To export ChatGPT data:
* Go to https://chatgpt.com/
* Open you account -> Settings -> Data controls -> Export data -> Export -> Confirm Export
	* All archived chats will be exported as well, so you might want to cleandelete them first
